# Getting Started with Create React App
```
npx create-react-app .
npm start
```
- HMR : Hot Module Replacement
- JSX(Javascript XML) : 자바스크립트 내부에 HTML문법과 비슷한 문법으로 작성하는 것이다.
- 리액트로 만들어진 페이지는 콤퍼넌트(Component)로 구성된다. 콤퍼넌트를 만들어서 결합해서 사용한다.
